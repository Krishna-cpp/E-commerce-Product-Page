TechKart - Product Page (Smartwatch Pro X)
==========================================

Project Description:
-------------------
This is a responsive e-commerce product page for "Smartwatch Pro X" built using HTML, CSS, and Bootstrap 5. 
It showcases product images, specifications, customer reviews, and a related products section.
The design follows a Flipkart-inspired theme with yellow and blue accents.

Features:
---------
1. Product Carousel:
   - Displays front, side, and back images of the smartwatch.
   - Responsive and interactive using Bootstrap carousel.

2. Product Details:
   - Title, description, price, and key features.
   - Add to Cart button triggers a Bootstrap modal.

3. Tabs Section:
   - Overview, Specs, and Details tabs.
   - Uses Bootstrap nav-tabs for interactivity.

4. Customer Reviews:
   - Displays user avatars, names, ratings, and comments.
   - Star ratings styled in blue.

5. Related Products:
   - Horizontal scrollable related products section.
   - Desktop arrows for easy navigation.

6. Responsive Design:
   - Fully responsive on mobile, tablet, and desktop devices.

Technologies Used:
------------------
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome 6
- JavaScript (for scrolling related products)
- Optional jQuery (can be removed if not needed)

How to Use:
-----------
1. Clone or download the repository.
2. Make sure the `images/` folder is in the same directory as `index.html`.
3. Open `index.html` in a web browser to view the product page.
4. Optional: Replace placeholder images in the `images/` folder with your own images.
